## Default Keycodes

These are default key codes that are used to map characters used in layout blocks to QMK variable names. You may declare your own in your own project and even overwrite these values if necessary.

### Symbols

```keycodes
- MINUS
= EQUAL
[ LBRACKET
] RBRACKET
\ BSLASH
; SCOLON
' QUOTE
` GRAVE
, COMMA
. DOT
/ SLASH
```
